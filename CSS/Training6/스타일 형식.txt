css소스 경량화 (주석, 줄바꿈, 공백등을 제거해 파일을 작게 만드는 것)
"css compress"사용해 파일을 작게.


* {속성 : 값; .......}
: 이 스타일을 모든 요소에 사용하겠다는 뜻이다.

태그명 { 스타일 규칙 }
: 특정 태그에 이 스타일을 적용하겠다는 뜻이다.

.클래스 명 { 스타일 규칙 }
:특정 부분만 선택해서 스타일을 적용하겠다는 뜻이다.

#아이디명 { 스타일 규칙 }
:클래스와 비슷한데 이는 중복해서 사용불가.

선택자1, 선택자2 { 스타일 규칙 }
:   이 스타일을 두 선택자 요소에 사용하겠다는 뜻이다.
