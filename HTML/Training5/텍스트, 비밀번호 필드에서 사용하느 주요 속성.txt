종류                                    설명
size            텍스트와 비밀번호 필드의 길이를 지정합니다. 즉, 화면에 몇글자가 보이도록 할 것인지를 지정합니다.
                예를 들어 최대로 입력할 수 있는 글자 수가 10개인데 size 속성값을 5로 지정하면 
                필드크기는 5개 글자 크기에 맞추고 나머지 5개 글자는 화면에 보이지 않습니다.
value           텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 보여주는 내용입니다.
                이 속성을 사용하지 않으면 빈 텍스트 필드가 표시됩니다. 비밀번호 필드에서 사용하지 않는 속성입니다.
maclength       텍스트 필드와 비밀번호 필드에 입력할 수 있는 최대 문자 수를 지정합니다.
